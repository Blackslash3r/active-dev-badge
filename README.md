---
## 🚀 Deployment

This bot runs automatically using **GitHub Actions**.

To deploy:
1. Fork this repo
2. Add your bot credentials as Secrets:
   - `TOKEN`
   - `CLIENT_ID`
3. Go to `Actions` tab > Click `run the bot` > Click `Run Workflow`
